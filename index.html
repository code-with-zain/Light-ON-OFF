Agar isON 0 hai, to bulb ko yellow color diya jata hai (indicating itâ€™s on), aur isON ko 1 set kar diya jata hai. Agar isON 1 hai, to bulb ka color transparent (off) ho jata hai, aur isON ko 0 set kiya jata hai. Is tareeke se, har button click se bulb ka state toggle hota hai, on ya off hota hai, aur isON variable us state ko reflect karta hai.

margin: 0 auto; ka matlab ye hai ke hum element (jaise 'div') ko horizontally center me lana chahte hain.
  • 0 ka matlab hai ke top aur bottom margin 0 pixels hogi, yani element ke upar aur neeche koi space nahi hoga.
  • auto ka matlab hota hai ke left aur right side ka margin automatically set ho, taake element center me aa jaye.

    • Asaan tareeqa: Jab hum margin: 0 auto; lagate hain, to browser left aur right ka equal space banata hai, jisse element beech me aa jata hai. Lekin ye tab kaam karega jab aapne element ki width set ki ho.
